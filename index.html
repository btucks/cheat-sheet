8 Easy Steps for Starting a New HTML Project

Description Example
Step 1 Create a Folder for the Project
Step 2 Open Preferred Code Editor
Step 3 Open a New Project with the Previously Made Folder in Step 1
Step 4 Add a File with the Name Index.HTML
Step 5 At the Beginning of the HTML File Write a Declaration for the Document Type <!DOCTYPE html>
Step 6 Add Opening & Closing HTML tags after Declaring Doctype <html> </html>
Step 7 Add Opening & Closing Head tags between HTML tags <head> </head>
Step 8 Add Opening & Closing Body tags between HTML tags and After Head tags <body> </body>
Congrats! You are now Ready to Begin a new HTML Project
